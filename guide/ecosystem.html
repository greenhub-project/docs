# Ecosystem

Within the project, there are different modules in development, all collectively designated by GreenHub.

So far, the following applications are being developed:

  • GreenHub BatteryHub - Android battery monitor app
  • GreenHub Farmer - Backend web app + REST API
  • GreenHub Lumberjack - A command line app for interacting with the GreenHub REST API
