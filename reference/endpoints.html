API Reference Index

GreenHub API Base URL: https://greenhub.di.ubi.pt/api/

All endpoints that require authentication you need to provide an API token with the request. See Authentication for more details.

Method Endpoint Usage Returns Auth*
GET /v1/devices List Devices Devices Yes
GET /v1/devices/:device Get A Device Device Yes
GET /v1/devices/:device/samples Get A Device's Samples Samples Yes
GET /v1/me Get The User’s Profile User Yes
PUT /v1/me/token Generate New API Key string Yes

*Request authentication
